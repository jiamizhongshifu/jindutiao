## 🔒 关于安全性

### 杀毒软件误报

Gaiya 使用PyInstaller打包（Python应用标准方式），部分杀毒软件可能误报。

**v1.4已优化**：
- ✅ 禁用UPX压缩 - 减少50-70%误报
- 🟡 白名单申请中 - 预计1-2周通过
- 📖 完全开源 - 所有代码可审计

### 如何安全使用？

**推荐方法：添加到杀毒软件信任列表**

**Windows Defender：**
1. 打开"Windows 安全中心"
2. "病毒和威胁防护" → "管理设置"
3. "排除项" → "添加排除项" → 选择文件
4. 选择 `Gaiya-v1.5.exe`

**360安全卫士：**
1. 打开360安全卫士 → 设置 → 信任与阻止
2. 信任区 → 添加信任文件
3. 选择 `Gaiya-v1.5.exe`

**火绒安全：**
1. 打开火绒 → 菜单 → 信任区
2. 添加文件 → 选择 `Gaiya-v1.5.exe`

**详细指南**：
- [安装教程](https://github.com/jiamizhongshifu/jindutiao/blob/main/SECURITY.md#-如何安全使用)
- [误报解决方案](https://github.com/jiamizhongshifu/jindutiao/blob/main/反病毒误报解决方案.md)

## 🎉 关于Gaiya

### 为什么叫Gaiya？

Gaiya这个名字有双重含义：

1. **希腊神话**：Gaia（盖亚）是大地女神，象征时间的承载者
2. **网络文化**：源自"哎呀"的谐音，代表时间管理的觉醒时刻

就像奥特曼变身一样，Gaiya帮你从"哎呀，时间又没了"变身为时间管理高手！

### Logo含义

🌍 地球/大地元素：象征Gaiya女神
⏱️ 时间守护：核心功能
⚡ 变身光效：隐藏彩蛋（懂的都懂）
