# GaiYa开源：用Python+PySide6打造的桌面时间管理工具

大家好，我是GaiYa的作者。今天很高兴在Product Hunt全球发布这个项目！

## 项目背景

作为程序员，经常在coding时失去时间感。一抬头天黑了，完全不知道这一天怎么过的。

市面上时间追踪工具很多，但都是**事后统计**型。我想要的是**实时感知**：让时间流逝像进度条一样清晰可见。

于是有了GaiYa（盖亚，希腊神话大地女神，象征时间永恒流动）。

## 核心功能

### 1. 透明桌面进度条
使用Qt的`setWindowFlags(Qt.FramelessWindowHint | Qt.WindowStaysOnTopHint)`实现透明置顶。

实时显示：
- 今日进度（当前时刻/24小时）
- 本周进度（周一至今/7天）
- 本月进度（本月1日至今/总天数）
- 本年进度（1月1日至今/365天）

### 2. AI任务规划
接入Claude 3.5 Sonnet API，输入今日目标，AI自动拆解为任务清单。

技术实现：
```python
import anthropic

client = anthropic.Anthropic(api_key=API_KEY)
response = client.messages.create(
    model="claude-3-5-sonnet-20241022",
    messages=[{"role": "user", "content": f"请拆解任务：{user_goal}"}]
)
```

### 3. 番茄工作法
自定义时长（15/25/45/60分钟），使用QTimer实现倒计时。

## 技术栈

- **前端框架**: PySide6 (Qt6 Python绑定)
- **AI引擎**: Claude 3.5 Sonnet API
- **打包工具**: PyInstaller
- **后端**: Vercel Serverless Functions (Python)
- **数据库**: Supabase (PostgreSQL)

## 开发难点与解决方案

### 难点1: PyInstaller打包Qt应用
**问题**: 打包后Qt样式表不生效

**解决**: 使用`--add-data`显式包含资源
```bash
pyinstaller --add-data "resources;resources" main.py
```

### 难点2: 透明窗口与始终置顶
**问题**: Windows 11 Acrylic效果冲突

**解决**: 使用`setAttribute(Qt.WA_TranslucentBackground)`
```python
self.setAttribute(Qt.WA_TranslucentBackground)
self.setWindowFlags(Qt.FramelessWindowHint | Qt.WindowStaysOnTopHint)
```

### 难点3: AI配额管理
**问题**: 免费用户滥用API导致成本过高

**解决**: Supabase记录使用次数，每日重置
```python
# 检查今日配额
quota_left = get_user_quota(user_id)
if quota_left <= 0:
    return {"error": "今日配额已用完"}
```
