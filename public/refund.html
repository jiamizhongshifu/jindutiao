退款政策

最后更新：2025年12月22日 | 生效日期：2025年12月22日

我们重视您的满意度。本退款政策说明了 GaiYa每日进度条会员订阅的退款条件、流程和时间。我们承诺公平、透明地处理所有退款请求。

1. 退款政策总览

GaiYa每日进度条提供灵活的退款政策，以确保您的消费权益。请仔细阅读以下退款规则：

情况 是否退款 退款金额 处理时间
支付成功但会员未到账 ✅ 支持退款 全额退款 7个工作日
重复支付 ✅ 支持退款 退还多余支付金额 7个工作日
技术问题导致无法使用 ✅ 技术支持或退款 按未使用时长比例退款 3-5个工作日审核
购买后7天内未使用 ✅ 支持退款 全额退款（扣除支付手续费） 5个工作日

会员期限已使用超过50% ❌ 不支持退款 - -
违反用户协议导致封禁 ❌ 不支持退款 - -

2. 可退款情况详解

2.1 支付成功但会员未到账

如果您完成支付但会员权益未在24小时内到账，可以申请全额退款或重新激活会员。

申请条件：

  • 提供支付凭证（订单号、支付截图、交易记录）
  • 确认账户邮箱正确且未绑定其他账户

处理时间：我们将在3个工作日内核实，7个工作日内完成退款。

2.2 重复支付

如果因系统错误或操作失误导致重复支付，我们将退还多余的支付金额。

申请条件：

  • 提供所有支付记录和订单号
  • 证明属于重复支付（非主动购买多个会员）

处理时间：7个工作日内完成退款。

2.3 技术问题导致无法使用

如果因软件 bug、兼容性问题等技术原因导致您无法正常使用会员功能，我们将优先提供技术支持。如果问题无法解决，可申请退款。

申请条件：

  • 提供详细的问题描述和错误截图/日志
  • 已尝试我们提供的故障排除步骤
  • 问题确实由我们的技术缺陷引起，而非用户环境问题

退款金额：按未使用时长比例计算。例如，月度会员使用10天后申请退款，退款金额 = 订阅费用 × (20/30)。

处理时间：技术审核3-5个工作日，退款7-15个工作日到账。

2.4 购买后7天内未使用

如果您在购买后7天内未激活或使用任何会员功能，可以申请全额退款（扣除支付平台手续费约3%）。

申请条件：

  • 购买时间不超过7天
  • 未使用 AI 任务生成、云同步等会员专属功能
  • 账户无异常或违规行为

处理时间：5个工作日审核和退款。

3. 不可退款情况

3.1 已激活并使用会员功能

一旦您使用了以下任一会员功能，即视为已激活会员权益，不支持因"不满意"或"改变主意"而退款：

  • 使用 AI 任务生成功能（消耗配额）
  • 启用云同步并上传数据
  • 导出数据报告
  • 使用高级主题或定制功能

3.2 会员期限已使用超过50%

如果您已使用超过订阅期限的50%（如月度会员超过15天、年度会员超过6个月），不支持退款。这是为了防止滥用退款政策。

3.3 违反用户协议

如果您的账户因违反 用户服务协议 而被暂停或终止，不支持退款。违规行为包括但不限于：

  • 恶意攻击服务器或滥用 API
  • 共享账户给他人使用
  • 使用非法手段获取会员权益
  • 其他违反法律法规或协议的行为

原路退回

4.1 第一步：提交退款申请

请通过以下方式提交退款申请：

  • 电子邮件：发送至 support@gaiyatime.com
  • 邮件标题：「退款申请 - 订单号 [您的订单号]」

邮件内容应包括：

  • 您的账户邮箱
  • 订单号或交易流水号
  • 退款原因（详细说明）
  • 支付凭证（截图或支付平台订单号）
  • 如果是技术问题，请附上错误截图或日志

4.2 第二步：审核处理

我们将在3-5个工作日内审核您的退款申请。审核过程中，我们可能会：

  • 核实您的支付记录和会员使用情况
  • 联系您了解更多信息或提供技术支持
  • 确认退款是否符合政策条件

审核完成后，我们将通过邮件通知您审核结果。

4.3 第三步：退款到账

一旦审核通过，退款将原路返回您的支付账户：

  • 支付宝/微信支付：7-15个工作日到账
  • 信用卡/借记卡：10-20个工作日到账（取决于银行处理速度）
  • PayPal/Stripe：5-10个工作日到账

退款到账时间受支付平台和银行处理速度影响，我们无法加速此过程。

5. 特殊情况处理

5.1 部分退款

在以下情况下，我们可能提供部分退款：

  • 年度会员在使用不到3个月时申请退款，按剩余月份比例退款
  • 技术问题导致部分功能不可用，按影响程度退款

5.2 替代方案

对于不符合退款条件的情况，我们可能提供以下替代方案：

  • 延期服务：将剩余会员时长延期到下次订阅
  • 降级方案：将专业版会员降级为基础版，差价退款
  • 技术支持：提供一对一技术支持解决问题

5.3 争议解决

如果您对退款决定不满意，可以：

  1. 在收到拒绝通知后7天内提出申诉，说明理由
  2. 我们将在5个工作日内重新审核
  3. 如果仍有争议，可以通过第三方支付平台的争议解决机制处理

6. 自动续费取消

如果您不想继续订阅，可以随时取消自动续费：

  • 在软件内的"账户设置"中关闭自动续费
  • 或通过支付平台（支付宝/微信/银行）取消订阅

重要提示：取消自动续费不会退款当前订阅周期的费用，但会阻止下一周期的自动扣款。

7. 联系我们

如果您对退款政策有任何疑问或需要帮助，请联系我们：

  • 电子邮件：support@gaiyatime.com
  • 邮件标题：「退款咨询 - [您的订单号]」
  • GitHub Issues：提交问题
  • 官方网站：www.gaiyatime.com

我们的客服团队将在1-2个工作日内回复您的咨询。

感谢您对 GaiYa每日进度条的支持！我们致力于提供公平、透明的退款政策，保障您的消费权益。如有任何问题，欢迎随时联系我们。
