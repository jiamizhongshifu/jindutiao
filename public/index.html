常见问题

快速解答您的疑问

💡 免费版和付费版有什么区别？

−

免费版可以永久使用核心功能（进度条、番茄钟、主题自定义），每天有 3 次 AI 智能规划机会，进度条带有品牌水印。 付费版可解锁更多 AI 配额（20-50次/天）、去除水印、数据云同步、场景系统、统计分析等高级功能。

💳 支持哪些支付方式？

−

目前支持微信支付和支付宝（国内用户推荐）。 Stripe 国际支付正在准备中，即将上线，届时支持 Visa/Mastercard 等国际信用卡。

🔄 购买后如何激活会员？

−

支付成功后，会员权益将自动激活，无需手动输入激活码。 只需重启应用即可享受会员功能，包括去除水印、增加AI配额等。

🛡️ 为什么杀毒软件报毒？

+

这是误报。GaiYa 使用 PyInstaller 打包，部分杀毒软件会将其标记为"未知应用"。 我们的软件完全开源，您可以在 GitHub 查看完整源代码，不包含任何恶意代码。 解决方法：将 GaiYa 添加到杀毒软件白名单即可。

💻 支持 Mac 或 Linux 吗？

+

目前 GaiYa 仅支持 Windows 10/11 (64位) 平台。 我们正在评估跨平台版本的可行性，敬请期待未来的更新。

🔙 可以退款吗？

+

GaiYa每日进度条是虚拟产品，一经购买激活后不支持退款。 购买前请仔细阅读功能介绍，建议先下载免费版体验核心功能后再决定是否购买会员。
